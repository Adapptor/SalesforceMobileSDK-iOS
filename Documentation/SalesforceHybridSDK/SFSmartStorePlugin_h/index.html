cursorByCursorId:isGlobal:

        isGlobal:(BOOL)isGlobal;
Parameters
cursorId

The unique ID of the cursor.

isGlobal

YES - if global store, NO - otherwise.

Return Value

The cached SFStoreCursor isntance with the given ID, or nil.

Discussion

Gets a handle to a cursor, from its ID.
