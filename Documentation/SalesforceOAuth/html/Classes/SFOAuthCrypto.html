Inherits from NSObject

Overview

Provides a wrapper for cryptographic services.

Warning: Once the decryptData or finalizeCipher method is called this object cannot be reused.

Instance Methods

decryptData:

Returns a decrypted representation of the supplied data if the data can be decrypted using the current key. Returns nil if decryption fails. Decryption is performed on the current thread.

- (NSData *)decryptData:(NSData *)data

Parameters

data

Encrypted data to decrypt.

Discussion

Warning: After this method is called this object must no longer be used.
