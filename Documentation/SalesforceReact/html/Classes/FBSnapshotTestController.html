FBSnapshotTestController Class Reference

Inherits from NSObject
Declared in FBSnapshotTestController.h

Overview

Provides the heavy-lifting for FBSnapshotTestCase. It loads and saves images, along with performing the actual pixel- by-pixel comparison of images. Instances are initialized with the test class, and directories to read and write to.
