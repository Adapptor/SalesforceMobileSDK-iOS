– applyFillColor:

For certain brushes we can fast path a combined fill and stroke. For those brushes we override applyFillColor which sets the fill color to be used by those batch paints. Those return YES. We can’t batch gradient painting in CoreGraphics, so those will return NO and paint gets called instead. @abstract
