– markStartForTag:

Starts measuring a metric with the given tag. Overrides previous value if the measurement has been already started. If RCTProfile is enabled it also begins appropriate async event. All work is scheduled on the background queue so this doesn’t block current thread.

- (void)markStartForTag:(RCTPLTag)tag

– markStopForTag:

Stops measuring a metric with given tag. Checks if RCTPerformanceLoggerStart() has been called before and doesn’t do anything and log a message if it hasn’t. If RCTProfile is enabled it also ends appropriate async event. All work is scheduled on the background queue so this doesn’t block current thread.
