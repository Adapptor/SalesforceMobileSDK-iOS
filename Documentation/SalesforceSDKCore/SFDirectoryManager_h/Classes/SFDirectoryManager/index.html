Introduction

Global directory manager that returns scoped directory. The scoping is enforced by taking into account the organizationId, the userId and the communityId.

The general structure follows this general template: <NSSearchPathDirectory> <-- For example, NSCachesDirectory will return Library/Caches <bundleIdentifier> <-- For example, com.salesforce.chatter <orgId> <userId> <internal> : internal community or base org [<communityId>]* : zero or more community specific folder

directoryForOrg:user:community:type:components:

- (NSString*)directoryForOrg:(NSString*)orgId user:(NSString*)userId 
        community:(NSString*)communityId type:(NSSearchPathDirectory)type 
        components:(NSArray*)components;
Parameters
orgId

The organization ID. If nil, this method returns the global directory type requested (eg Library/Caches)

userId

The user ID. If nil, this method returns the directory type requested, scoped at the org level (eg Library/Caches/<orgId>/)

communityId

The community ID. If nil, this method returns the directory type requested, scoped at the user level (eg Library/Caches/<orgId>/<userId>)

type

The type of directory to return (see NSSearchPathDirectory)

components

The additional path components to be added at the end of the directory (eg ['mybundle', 'common'])

Return Value

The path to the directory

Discussion

Returns the path to the directory type for the specified org, user and community.
