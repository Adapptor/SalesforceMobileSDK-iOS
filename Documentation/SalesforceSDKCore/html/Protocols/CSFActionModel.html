CSFActionModel Protocol Reference

Conforms to NSObject
Declared in CSFActionModel.h

Overview

This protocol represents objects that can be used as the model class for a CSFAction.

When a network request is performed, if the [CSFAction modelClass] property is set to a class that conforms to this protocol, the action will attempt to construct a model with the result of the network request.

– initWithJSON:context:

Designated initializer to construct a model object from its JSON representation.

- (id)initWithJSON:(id)json context:(NSDictionary *)context

Parameters

json

Dictionary of structured data from the network.
context

Dictionary of relevant information about the request and the action that performed it.

Return Value

Initialized model object.
