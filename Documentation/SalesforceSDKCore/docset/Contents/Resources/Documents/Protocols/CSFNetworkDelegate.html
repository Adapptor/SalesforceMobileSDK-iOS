– network:willEnqueueAction:

Called when an action is about to get enqueued.

- (void)network:(CSFNetwork *)network willEnqueueAction:(CSFAction *)action

Discussion

@discussion This method is called synchronously, giving the delegates a chance to inspect the action before CSFNetwork acts on it. Perform as little work as possible on this callback to avoid performance degradation.
