deleteAccountForUserId:error:

Allows you to remove a user account associated with the given user ID.

- (BOOL)deleteAccountForUserId:(NSString *)userId error:(NSError **)error

Parameters

userId

The User ID of the account to remove.

error

Output error parameter, populated if there was an error deleting the account (likely from the filesystem operations).

Return Value

YES if the deletion was successful, NO otherwise. Note: If no account matching the userId parameter is found, no action will be taken, and deletion will be reported as successful.
